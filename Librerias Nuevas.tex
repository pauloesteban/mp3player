\textbf{Proceso para la instalación de las Librerías en mp3payer. }
\vspace{3ex}

\textbf{descargar: }
descargamos el contenido que se encuentra en el repositorio mp3player el cual es jlgui3.0.zip 
Link de descarga : https://github.com/pauloesteban/mp3player


\includegraphics[scale=0.5]{1.png} 

\textbf{descomprimir }

damos clic derecho a la archivo jlgui3.0.zip y descomprimimos el contenido en la carpeta peppermusic de java 

\includegraphics[scale=0.5]{2.png} 



\textbf{agregar librerías }

abrimos el NetBeans y nos vamos a la carpeta PepperMuisc en la cual le vas a dar a + para que se despliegue todo el contenido de la carpeta, nos dirigimos a Libraries en donde daremos clic derecho para desplegar la opciones de las cuales escogeremos Add JAR/Folder          

\includegraphics[scale=0.5]{3.png} 


	\vspace{1ex}

se nos desplegara una ventana en la cual buscaremos donde guardamos la carpeta descomprimida (jlgui3.0.zip), en donde se mostrara la primera librería que vamos a ingresar la cual es jlgui3.0.jar en donde le daremos a abrir para poderla ingresar.

\includegraphics[scale=0.5]{4.png} 

\vspace{1ex}
aremos el mismo procedimiento pero ahora vamos a buscara la carpeta "Lib" en la cual abriremos y se mostrara una serie de librerías que usaremos así como ingresamos la primera librería aremos lo mismo para ingresar todas esa librerías que son necesarias para el programa mp3player.
