Operaciones de los botones del Reproductor Multimedia

Esta aplicacion sirve para la reproducción de audio formato MP3.

Es un reproductor multimedia creado por la empresa SaltoSoft en el año 2017 para el sistema operativo Microsoft Windows. La versión actual es la última existente, que se incluye con Windows 7, Windows 8 y Windows 10. El reproductor tiene las mismas funcionalidades en todas las versiones.
